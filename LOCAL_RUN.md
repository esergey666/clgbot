# Локальный запуск и быстрая проверка

## Запуск Telegram-бота вручную

1. Заполните `.env`:

```env
bot=токен_бота
admins=ваш_telegram_id
```

2. Запустите файл:

```bat
run_bot.bat
```

Или из терминала:

```bat
.venv\Scripts\python.exe -m bot.main
```

## Быстрая проверка генерации без Telegram

Для проверки шаблона и правок можно сгенерировать PNG локально:

```bat
run_local_label.bat
```

Файл появится здесь:

```text
data\local_label.png
```

Можно подставить свои данные:

```bat
.venv\Scripts\python.exe -m bot.local_label --art 761530404 --color V0158 --size 30 --code TOM068804 --certilogo-code CLG047604293519 --certilogo-url https://certilogo.com/CLG047604293519 --output data\test.png
```

Для бирки 45 мм добавьте:

```bat
--type clg2026
```

## Про распознавание двух фото

Это можно добавить отдельным режимом в Telegram-боте:

1. пользователь выбирает бирку;
2. отправляет первое фото с артикулом, цветом, размером и внутренним кодом;
3. отправляет второе фото с Certilogo-кодом и QR;
4. бот показывает найденные значения для подтверждения;
5. после подтверждения генерирует бирку.

Для QR лучше использовать декодер QR/DataMatrix. Для текста с фото нужен OCR или Vision API. Самый надежный вариант для таких наклонных фотографий этикеток - Vision API, потому что обычный локальный OCR часто ошибается на бликах, поворотах и мелком шрифте.

## Как включить бесплатное распознавание фото

Распознавание работает локально: RapidOCR читает текст, Tesseract OCR используется как запасной вариант, OpenCV читает QR.
Для QR первым используется WeChat QR detector, он лучше обычного OpenCV на плохих фото.

Что нужно сделать один раз для запасного Tesseract:

1. Установите Tesseract OCR для Windows.
2. Лучше ставить в стандартную папку `C:\Program Files\Tesseract-OCR\`.
3. Перезапустите `run_bot.bat`.

Если Tesseract установлен в другую папку, добавьте в `.env` путь:

```env
TESSERACT_CMD=C:\путь\к\tesseract.exe
```

Как пользоваться:

1. Нажмите `Создать файл`.
2. Выберите `Бирка 40мм` или `Бирка 45мм`.
3. Отправьте первое фото с артикулом, цветом, размером и кодом.
4. Отправьте второе фото с Certilogo-кодом и QR.
5. Проверьте найденную строку и нажмите `Сгенерировать бирку`.

Если Tesseract не установлен, бот все равно попробует RapidOCR. Tesseract нужен только как дополнительный запасной OCR.
